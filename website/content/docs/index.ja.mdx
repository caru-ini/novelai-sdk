NovelAI の画像生成 API のための、モダンで型安全な Python SDK です。
Pydantic v2 による堅牢なバリデーションと完全な型ヒントを備え、開発者体験 (DX) を重視して設計されています。

## 主な機能

- **完全な型安全性**: Python 3.10+ 対応、Pydantic v2 による厳格なバリデーション。
- **高レベル API**: 直感的で使いやすいインターフェース。
- **最新機能への対応**: V4 モデル、精密参照 (キャラクター参照)、ControlNet などをサポート。
- **便利なユーティリティ**: PIL/Pillow 統合、SSE ストリーミングなど。

## データモデル・アーキテクチャ

このライブラリは、2 つの異なるデータモデル層で設計されています。

![モデルアーキテクチャ](/images/model-architecture.png)

1. **User Model (推奨)**: 適切なデフォルト値と自動バリデーションを備えた、ユーザーフレンドリーなモデル。
2. **API Model**: NovelAI の API エンドポイントと 1 対 1 で対応する、主に内部で使用されるモデル。
